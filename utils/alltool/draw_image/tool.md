## draw_image：简易画图工具

### 目标

根据文本提示生成一张简单 PNG 图片（白底 + 文本），用于快速占位、流程验证与文档插图。

### 输入

- prompt：图片内容描述（会被绘制为文本）
- save_path（可选）：保存路径（仓库内相对路径）；默认保存到 `docs/generated/` 下
- width/height（可选）：图片尺寸（像素）

### 输出

- 返回保存后的绝对路径与文件大小信息

### 安全与边界

- 只允许写入仓库目录内的文件
- 仅允许 `.png` 扩展名
- 尺寸做上限限制，避免生成过大的图片

### 使用方式

该工具包含 `tool.py`，可被装配到某个角色的 `tools/` 目录后由运行时加载并注册到 toolkit 中。
